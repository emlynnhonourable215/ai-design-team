## 🚀 What this app does

ai-design-team is a multi-role AI design team skill for research, planning, scripting, design, and content production. It helps you move from idea to finished work with less back and forth.

Use it to:

- Gather research notes
- Shape a plan from a rough idea
- Draft scripts and content
- Support design tasks
- Keep work in one simple flow

It is built for everyday users who want a clear path from first idea to final output.

## ✍️ How to use it

The app works like a small team of helpers. Each role focuses on one part of the task.

### Research
Use this role when you need background info, topic notes, or a quick scan of an idea.

### Planning
Use this role when you want a step-by-step plan, task list, or workflow.

### Scripting
Use this role for video scripts, voiceover text, scene flow, or short-form content.

### Design
Use this role when you need layout ideas, visual direction, or simple creative concepts.

### Content production
Use this role to turn notes into drafts, posts, page copy, or other ready-to-edit text.

A good way to work:

1. Start with research
2. Move to planning
3. Draft the script or content
4. Review the design direction
5. Edit the final output

## ⚙️ Tips for better results

Use short, clear prompts. For example:

- Create a research brief for a new app idea
- Build a 5-step launch plan for a local service
- Write a 60-second video script about healthy snacks
- Give me three design directions for a landing page
- Turn these notes into a clean article outline

If the result feels too broad, add more detail:

- Who the work is for
- What the goal is
- What tone you want
- How long the output should be
- What format you need

## 🔧 Troubleshooting

### The file will not open
- Check that the download finished
- Make sure you extracted the ZIP file first
- Try opening the app file again
- Move the file to a simple folder like Downloads

### Windows blocks the app
- Right-click the file and choose Properties
- If you see an Unblock option, turn it on
- Click More info on the Windows prompt
- Choose Run anyway if you trust the file

### The app opens but shows a blank screen
- Wait a few seconds for it to load
- Refresh the page if it runs in the browser
- Close and reopen the app
- Check your internet connection

### The output is not what you want
- Use a shorter prompt
- Add the audience and goal
- Ask for one role at a time
- Break large jobs into smaller steps
